# Patching Core Components

This folder contains the core CloudFormation templates for the Engage Patch Management solution.

## Components

### patching-main

The main template that deploys core resources in the audit/central account.

### patching-spoke-1-2

The spoke template that deploys resources in target accounts.

### patching-spoke-2-2

Similar to patching-spoke-1-2, but for additional accounts or regions.
